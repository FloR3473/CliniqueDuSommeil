## Prérequis

Avant de commencer, assurez-vous d'avoir installé :

- Node.js >= 20
- npm >= 10
- Git
- Angular CLI



## Branches

Ne travaillez pas directement sur main.

Créer une branche :
git checkout -b dev (par le lead)
git checkout -b nom_des_collab
git checkout -b feature/nouvelle-fonctionnalite


## Commits

Format :

feat: ajout d'une fonctionnalité

fix: correction d'un bug

docs: modification documentation


## Pull Request

Avant d'envoyer une PR :

- Tester le projet
- Vérifier le code
- Décrire les changements
